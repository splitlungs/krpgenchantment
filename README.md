# krpgenchantment
Warning: This is a Work In Progress 
Expect lots of game-breaking changes as development continues.
This is only a placeholder until the mod is completed.
If you use the mod right now, it's your fault.

Requires KRPGLib.

 

This mod aims to add a "Minecraft-like" style of Enchanting system.

Build an Enchanting Table, and apply magical enchantments to your items.

Includes a JSON Recipe system, so you can add your own items as magical reagents to apply any of the included enchantments to any item.
